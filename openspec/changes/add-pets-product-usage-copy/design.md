## Design

La información de uso debe aparecer cerca de la descripción de cada producto, no escondida en una tabla aparte. El objetivo es que la ficha responda en segundos a la pregunta práctica: "¿cómo se usa?".

### Home principal

- Mantener el bloque visual actual de cada tarjeta.
- Insertar una línea corta, escaneable y consistente bajo el texto principal.
- Usar un patrón común:
  - `Cómo se toma` para masticables y aceite.
  - `Cómo se aplica` para el spray.
  - `Cómo se usa` para el snack funcional.

### Guía de producto

- Repetir la misma información en el panel detallado de cada producto.
- Usar una nota breve para que la lectura siga siendo rápida.
- No añadir claims nuevos; solo trasladar instrucciones ya presentes en las fichas fuente.

### Source of truth

- Las fichas `.md` de `FICHAS USA PETS` son la referencia para la redacción de uso.
- La copia debe ser coherente entre `index.html` y `guia.html`.
